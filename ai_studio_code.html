Technical Specification

The centipede logic utilizes a chain-following algorithm where each segment targets the previous one's position with a fixed distance constraint. The Inverse Kinematics (IK) for the legs is calculated using the Law of Cosines across a 3-joint structure (Coxa-Femur-Tibia). Leg animations are driven by a metachronal wave pattern, simulating the natural gait of arthropods.
Each segment is rendered with a radial gradient to simulate chitinous shell depth. The ink-sketch aesthetic is achieved through fine line-work and a high-contrast palette.
  • Physics: Verlet Integration
  • Constraint: Segment Distance Locking
  • Input: Multi-touch pointer support
  • Visuals: Procedural Ink Hatching
